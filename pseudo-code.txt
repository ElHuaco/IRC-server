- Clases: Server, Clients, Commands (Virtual) de RFC1469, Canales.
	- Server son los FD_SET masters + array de Clients y Canales
	+ métodos de accept() y atributos para commands de server.
		- Server inicia socket(), bind(), listen() según getaddrinfo(argv).
		- Server hace freeaddrinfo();
		- Los Clients se añaden y quitan de Server según se accept()an o cierran.
		- Los Canales se crean y destruyen en el Server.
	- Clients son un socket + atributos para commands de user.
		- Operadores son clients con acceso a ciertos commands.
	- Canales son una lista de sockets + atributos para commands de canal.
